# Employee Database Analysis

## Project Overview
This project focuses on analyzing employee, department, and salary data using SQL to answer common HR and management business questions.

## Tools & Technologies
- MySQL
- SQL (JOINs, GROUP BY, HAVING, subqueries, aggregate functions)

## Database Structure
The database contains the following tables:
- employees
- departments
- dept_emp
- dept_manager
- salaries
- titles

Relationships are maintained using primary and foreign keys to ensure data integrity.

## Business Questions Solved
- Which employees belong to which departments?
- What is the average salary by department?
- Which departments have the highest salaries?
- Which employees have worked in multiple departments?
- Which employees received multiple salary changes?

## Files in This Repository
- `schema.sql` → Database schema with keys and relationships
- `queries.sql` → SQL queries used for analysis

## Key Learnings
- Writing complex multi-table JOIN queries
- Using aggregate functions for HR analytics
- Analyzing employee and salary trends using SQL
